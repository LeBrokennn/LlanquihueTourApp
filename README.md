# LlanquihueTourApp

Repositorio del proyecto de la agencia turística **Llanquihue Tour**, desarrollado en Java para la asignatura Desarrollo Orientado a Objetos I.

## Actividades

| Carpeta | Tema | Descripción |
|---------|------|-------------|
| [`actividad1-registros/`](actividad1-registros/) | Colecciones y CSV | Carga de personas desde archivo, filtros y búsquedas |
| [`actividad2-polimorfismo/`](actividad2-polimorfismo/) | Herencia y polimorfismo | Servicios turísticos con `List<ServicioTuristico>` |

Cada carpeta tiene su propio `README.md` con instrucciones de compilación y ejecución.

## Estructura del repositorio

```text
LlanquihueTourApp/
├── README.md
├── actividad1-registros/
│   ├── data/personas.csv
│   └── src/
└── actividad2-polimorfismo/
    └── src/
```

## Requisitos

- Java JDK 8 o superior
- IntelliJ IDEA (opcional)
